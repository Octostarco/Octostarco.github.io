Octostar Singlenode Installation

To get started with the installation of Octostar Singlenode, please ensure that you have obtained a Docker Hub token from Octostar. This token is required to proceed with the installation.

Replace dockerhub_token in the script below with the Docker Hub token you receive from Octostar. Once you have your token, run the following command in your terminal to start the installation process: 

curl https://octostarco.github.io/debian-arm64.sh | env DOCKERHUB_TOKEN=dockerhub_token bash
