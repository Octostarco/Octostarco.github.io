Octostar Singlenode Installation

To get started with the installation of Octostar Singlenode, please ensure that you have obtained a Docker Hub token from Octostar. This token is required to proceed with the installation.

Currently, the installation is available only for Debian systems.

Replace dockerhub_token in the script below with the Docker Hub token you receive from Octostar. Ensure that the user executing the script has root permissions or sudo access to run the installation successfully. 

curl https://octostarco.github.io/debian.sh | env DOCKERHUB_TOKEN=dockerhub_token bash

If you encounter any issues during installation, please contact our DevOps team at devops@octostar.com for assistance.
